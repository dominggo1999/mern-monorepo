## Minimal monorepo MERN stack boilerplate

### Development 
  - Start backend and frontend concurrently ```npm start```
  - Start backend : ```npm run server:dev```
  - Start frontend : ```npm run client:dev```

### Stack
  - Client 
    - Vite
    - React 
    - React router dom
    - Twin Macro
    - Styled Components
    - Zustand
    - Immer
  - Backend
    - Express
